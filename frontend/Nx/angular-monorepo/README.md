# Steps to create and run Nx Angular Monorepo Workspace

## Create Nx Workspace
- npx create-nx-workspace _workspace-name_ --preset=angular-monorepo --appName=_app-name_

## Bring Up Development Server
- npx nx run _app-name_:serve
